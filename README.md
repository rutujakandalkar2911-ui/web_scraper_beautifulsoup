## Project Name
Web Scraper using BeautifulSoup

## Project Scope
This project is a Python web scraper that uses Requests and BeautifulSoup to extract quotes from https://quotes.toscrape.com.

## Features
- Fetches webpage content
- Extracts quotes
- Displays quotes in the terminal
- Uses BeautifulSoup for HTML parsing

## Technologies Used
- Python
- Requests
- BeautifulSoup4

## Project Structure

```
Source_Code/
├── main.py
├── requirements.txt
```

## How to Run

```bash
pip install -r Source_Code/requirements.txt
python Source_Code/main.py
```

## Output
The program displays all quotes available on the webpage.
